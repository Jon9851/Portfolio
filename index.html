HTML, CSS & Javascript

Second Project

Project 2

Project 2 consists of html, CSS and JavaScript.
The project in design is quite simple as my main goal for the creation of this site was to get a good understanding on how JavaScript works.

The quiz of the world is a simple quiz game that contains 3 buttons on the home page. Each button will take user to three different pages. The buttons will be highlighted once the user clicked on them, this so that the user and keep track of which page they are on.

The game contains multiple different features which are designed to help the user while playing the quiz of the world. Located on the top left is a question counter to help the user keep track of what question they are up too.

Below the question counter is a progress bar which fills up as the user progress through the quiz. Located on the top right is a score counter which will help the user keep track of the scores. The questions are displayed below the progress bar and change at random once a question has been answered. The quiz it is self is a multiple-choice quiz with four possible answers with 17 questions in total. Each option will change colour once selected.
