    • Project 4

      Eshop is an e-commerce online store that offers a wide range of tech products for its users. The site is designed to be user-friendly with a simple navigation bar and dropdown menu that displays all available categories and products. The site is built using the Django framework and PostgreSQL to manage the inventory database, while Stripe payments provide secure checkout payment. The site also features a user profile that includes order history and a complete list of user purchases. Bootstrap 4 was used to create many of the site's features, and the color scheme is primarily purple and white with red serving as the warning color. The product page displays product cards in rows of four with a purple border, and each card includes edit and delete buttons, a category tag, price, and a brief description of the product. The homepage includes a search bar and a profile menu with a shopping bag, and there is also an image and a button that takes you to the online store's all products page. In addition to Django, Python3 is used to create the site's routes, while Balsamiq is used for wireframing. Gitpod and Github are used for deploying and hosting the site's code, respectively, and Code Beautify and W3 Schools are used for testing HTML and CSS code. JavaScript (ES6) is used for the site's functionality, and responsive design is tested on all devices using ami. Heroku is used for deployment, and MongoDB is used for the user database, while PostgreSQL is used for the publisher games and reviews database. PEP8 Online Checker is used to check Python routes, and Bootstrap4 is used to create HTML templates. Django_Allauth is used to provide security for the app, and Dropbox is used to store images before AWS deployment. Miro is used to create the eshop diagram, while Amazon Web Services is used on the deployed version to store static and image files. Finally, Stripe is used for card payments on the online store.
